Airline Passenger Satisfaction

Predicting the passenger experience from survey and service data — turning a satisfaction dataset into a working prediction tool.

Agrima Sharma · BIT 4.2C — Programming for Data Analytics · July 2026

The Problem

Airlines run satisfaction surveys constantly, but usually analyze them only after the fact — once a passenger has already churned. This project asks: can passenger satisfaction be predicted from their profile and service ratings, reliably enough to act on?

The Data

129,880 passenger survey responses across 24 columns — demographics, trip details, and 0–5 service ratings, sourced from a public airline satisfaction survey dataset. Only one column (Arrival Delay) had missing values, cleaned via median imputation.

Approach

1. Clean & Explore

Missing value handling, duplicate checks, target balance, and distribution analysis across key variables.

2. Visualize

Histograms, bar charts, boxplots, and a correlation heatmap to surface relationships with satisfaction.

3. Test

Chi-square and t-tests confirming Class and Age are statistically significant satisfaction factors.

4. Model & Ship

Six classifiers compared; the best model powers an interactive Streamlit prediction tool.

Key Findings

  • Class matters most: Business class passengers are mostly satisfied; Economy passengers are overwhelmingly dissatisfied (χ² ≈ 32,906, p < 0.001).
  • Age and loyalty: Satisfied passengers are, on average, significantly older, and returning customers skew older than first-time flyers (t ≈ 48.76, p < 0.001).
  • Model choice matters: Tree-based ensembles (XGBoost, Random Forest, AdaBoost) far outperform distance-based methods (KNN, SVC), which are sensitive to unscaled features.

The Product

Rather than stop at a static analysis, the trained model is wrapped in an interactive Streamlit app: enter a passenger's profile and service ratings to get a live satisfaction prediction, see which service areas show the biggest satisfaction gap, and view model-driven recommendations for where an airline should focus improvement efforts.
